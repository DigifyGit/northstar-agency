# NorthStar Hero Concept Lab — Antigravity

Fresh concept lab for the NorthStar Agency hero section.  
All concepts built by **Antigravity** (Google DeepMind).

## Locked Constraints

- No space/nebula/cosmic wallpaper
- No skyline, constellation dots, abstract swirls
- No generic dashboard-card repetition
- Module labels must be legible and named
- Verification gates must be visible and understandable
- Visual metaphor understandable in <3 seconds

## Product Language

**Modules**: Mission Control · Skill Recovery Lab · Case Command · Strategic Roadmap · Next 7 Days  
**Gates**: VERIFIED_LEGIT · VERIFIED_RISK · UNKNOWN_INSUFFICIENT_EVIDENCE
