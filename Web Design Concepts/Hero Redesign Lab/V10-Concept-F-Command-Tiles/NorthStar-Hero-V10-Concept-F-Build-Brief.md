# NorthStar Hero V10 - Concept F Build Brief

## Concept Name
Command Tiles Stack

## Objective
Represent product modules as tangible command assets with a clear structured workflow spine.

## Prompt (locked)
```text
4K, 16:9 ultra premium 2026 hero for “NorthStar”.
Hard reset: NO space, NO skyline, NO nebula, NO orbit arcs, NO constellation dots, NO glowing trails.
Background: deep navy #050914 with soft studio haze only, clean and positive.

Visual concept: COMMAND TILES STACK.
Composition: left 45 percent is a stacked set of five premium physical tiles, right 55 percent negative space for typography.

Top element: the precision 16-point compass rose North Star as a machined emblem embedded into the top tile (not floating).
Under it: 5 thick rectangular tiles (like premium hardware modules) slightly offset, each tile labeled with crisp engraved text:
MISSION CONTROL
SKILL RECOVERY LAB
CASE COMMAND
STRATEGIC ROADMAP
NEXT 7 DAYS
Each tile has a minimal status chip inset (not UI), with stamped words:
VERIFIED, RISK, or IN REVIEW (small, industrial, not glowing).

Material language: frosted glass edges + titanium frames + satin anodized indigo accents #5F6AD9, micro teal accent #3FC8B4 used only as a tiny indicator LED.
Lighting logic: soft key from upper-left plus faint bounce fill from lower-right, clean premium product shot, no neon.

Meaning layer: one thin precision “link spine” connects the tiles vertically like a data backplane, implying structured workflow and lifecycle.
No lines that look like charts, no grids, no diagrams.

Right side UI overlay:
Navbar: NORTHSTAR | Platform | Methodology | Enterprise Solutions | Intelligence | Log In
Headline: Architecting the Future of Workforce (gradient only on “Future of Workforce” #7BC0D7 to #6E82CB)
Subtext: Precision Talent Intelligence. Verified Execution.
Buttons as before.

Negative: no sci-fi wallpaper, no particles, no cartoon icons, no people silhouettes, no wireframes.
```

## Variations
- A: tighter stack with minimal offset.
- B: stronger offset and visible backplane spine.

## QA checklist
- Tiles read as physical modules, not UI cards.
- Lifecycle flow is understandable without copy overload.
- Status chips are legible but restrained.
