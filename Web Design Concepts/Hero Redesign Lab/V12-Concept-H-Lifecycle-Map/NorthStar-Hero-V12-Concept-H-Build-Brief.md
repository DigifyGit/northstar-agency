# NorthStar Hero V12 - Concept H Build Brief

## Concept Name
Inlaid Lifecycle Map

## Objective
Present lifecycle modules as a readable, premium physical inlay system with optimistic enterprise clarity.

## Prompt (locked)
```text
4K, 16:9 ultra premium 2026 hero for “NorthStar”.
Hard reset: NO space, NO skyline, NO nebula, NO swirls, NO trails, NO dot networks.
Background: deep navy #050914 with a subtle warm-to-cool studio gradient (still corporate), clean and optimistic.

Visual concept: INLAID LIFECYCLE MAP.
Composition: left 45 percent is a single large rectangular slab like a premium desk surface insert, right 55 percent negative space.

On the slab: an inlaid circular track (not a chart) with 5 engraved bays.
In the center bay: the precision 16-point compass rose North Star emblem, inlaid flush, brushed titanium.
Around it: 5 bays labeled with crisp engraved titles:
MISSION CONTROL
SKILL RECOVERY LAB
CASE COMMAND
STRATEGIC ROADMAP
NEXT 7 DAYS
Each bay contains a minimal inset “artifact” shape (tiny physical object), representing deliverables, not icons.

Lighting: soft key upper-left, gentle rim light along slab edge, clean and bright, no glow.
Material language: dark ceramic slab, titanium inlays, frosted glass inserts, micro indigo accent on North tip.

Right side UI overlay standard.
Negative: no UI grids, no tech spirals, no sci-fi scenery, no people.
```

## Variations
- A: darker slab, stronger inlay contrast.
- B: brighter optimistic slab with softer edges.

## QA checklist
- Inlay bays must be clearly labeled.
- Must feel physical and premium, not UI diagrammatic.
- Preserve clean optimistic tone.
