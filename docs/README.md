# Documentation Layout

This folder contains operational documentation that should not live in project root.

## Subfolders

- `handover/`: migration-grade introductions and emergency handoff docs.
- `plans/`: transformation plans and implementation playbooks.
- `imports/gpt_web/`: imported artifacts from ChatGPT Web with indexing metadata.
- `client_assets/`: candidate assets (CV variants, notes) when not treated as archive items.
- `snippets/`: reusable prompt blocks for consistent agent behavior.
