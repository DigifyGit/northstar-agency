# Snippet Shortcuts

Use these keyword triggers in your text expander, clipboard manager, or IDE snippet tool.

## Recommended Shortcuts

- `/nsmark` -> paste `docs/snippets/AGENT_MARK_SESSION_SNIPPET.md` *(activates Mark persona session)*
- `/nslux` -> paste `docs/snippets/NORTHSTAR_LUXURY_AGENCY_SNIPPET.md`
- `/nsreport` -> paste `docs/snippets/NORTHSTAR_LUXURY_REPORT_ADDON.md`
- `/nsfresh` -> paste `docs/handover/FRESH_SESSION_KICKOFF_PROMPT.md`

## Setup Note

This repo stores snippet content only. Shortcut expansion itself is configured in your local tool (for example Raycast snippets, Espanso, TextExpander, Keyboard Maestro, or IDE snippets).
