# NorthStar Internal - Follow-up Sequence

## Purpose
Standardize post-application follow-up timing and message intent.

## Day 1 - Thank-you / confirmation
- Objective: confirm interest and professionalism.
- Message points: appreciation, role reference, availability.

## Day 7-10 - Status check
- Objective: request update without pressure.
- Message points: role + application date + concise status ask.

## Day 21 - Final close-loop follow-up
- Objective: close loop respectfully and keep future consideration open.
- Message points: continued interest, request final status, professional close.
